Educational Tool for 5G Power Consumption Modelling

This project presents a software application for 5G NR Power Modeling developed in MATLAB. The tool includes an interactive GUI enabling users to compare power consumption across 5G architectures: D-RAN, C-RAN, and Cloud C-RAN.
